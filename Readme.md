# Dump PDF/UA (tagged PDF) structure to XML

Run

    pdfuaanalyze myfile.pdf

which creates an XML tree of the tagged PDF structure, for example

```xml
<Document>
  <Art>
    <H1></H1>
    <P>
      <Link></Link>
      <Link></Link>
    </P>
  </Art>
</Document>
```
